# 移动端组件

## 描述

-  音乐播放器
-  手机拍照
<br>
其他说明：<br>
1、调用摄像头必须是 https 协议，当前在启动脚本内已经配置<br>
2、IE浏览器和部分非主流浏览器不支持调用摄像头，内部已判断不支持则提示<br>
-  瀑布流
-  加载状态
-  通知栏
-  拖动条
-  多文本折叠
-  图片上传
-  触底加载
